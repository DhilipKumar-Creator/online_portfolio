## ✨ Features

- 🌙 **Premium Dark Theme** — Sleek design with purple-cyan accent gradients
- ⚡ **Smooth Animations** — Scroll-reveal effects, typing animation, micro-interactions
- 📱 **Fully Responsive** — Optimized for all screen sizes (mobile, tablet, desktop)
- 🎨 **Glassmorphism Design** — Modern cards with glass-like effects and glowing borders
- 🖼️ **Animated Hero Section** — Typing effect, floating badges, gradient profile border
- 📂 **Project Showcase** — Detailed project cards with tech stack tags
- 📜 **Education Timeline** — Visual timeline of academic journey
- 🏆 **Certifications Section** — Badge-style credential display
- 📬 **Contact Section** — Contact info with email integration

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Structure & Semantics |
| CSS3 | Styling, Animations, Responsive Layout |
| JavaScript | Interactivity, Scroll Effects, Typing Animation |
| Google Fonts | Typography (Inter, Outfit, JetBrains Mono) |
| Lucide Icons | UI Icons |
